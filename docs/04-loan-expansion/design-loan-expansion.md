## 1. 배경 및 목적

현재 대출 상품이 4개 은행 + 7개 공공 상품(총 11개)으로 매우 제한적.  
실제 사용자가 비교·선택할 수 있도록 **시중 12개 은행 + 정부/공공 상품 총 36개**로 확장하고,  
정렬·검색·필터 기능을 추가하여 사용성을 대폭 향상시킨다.

## 2. 타입 변경 사항

### LoanCategory 확장

```typescript
// shared/types/loan.ts
export type LoanCategory = 'bank' | 'housing' | 'government' | 'public';
//                                                               ^^^^^^ 신규: LH/SH

export type InterestRateType = 'variable' | 'fixed' | 'mixed';
// variable = 변동, fixed = 고정, mixed = 혼합

export type TargetGroup = 'general' | 'youth' | 'newlywed' | 'multi-child' | 'senior';
// general=일반, youth=청년, newlywed=신혼부부, multi-child=다자녀, senior=고령자

export interface LoanProduct {
  id: string;
  category: LoanCategory;
  name: string;
  provider: string;
  interestRate: number;         // 기존 유지 (대표 금리 = min 기준)
  interestRateMin: number;      // 신규: 최저 금리
  interestRateMax: number;      // 신규: 최고 금리
  interestRateType: InterestRateType; // 신규: 금리 유형
  maxAmount: number;
  minPeriod: number;
  maxPeriod: number;
  repaymentMethods: RepaymentMethod[];
  targetGroups: TargetGroup[];  // 신규: 대상 그룹 (복수 가능)
  description: string;
  conditions: string[];
  url?: string;                 // 신규: 공식 링크 (선택)
}
```

## 4. UI 개선 사항

### 4-1. 탭 구조 변경

```
기존: [은행] [주택공사] [정부]
신규: [전체] [은행] [정부기금] [공공임대]
```

카테고리 매핑:
- `bank` → 은행 탭
- `government` → 정부기금 탭  
- `housing` + `public` → 공공임대 탭

### 4-2. 검색 + 정렬 + 필터 바

탭 아래에 컨트롤 바 추가:

```
┌─────────────────────────────────────────────┐
│ 🔍 [은행명 검색...          ]  [정렬▼]      │
│ [전체] [청년] [신혼부부] [다자녀] [고령자]   │
└─────────────────────────────────────────────┘
```

**정렬 옵션 (Select 드롭다운)**:
- 금리 낮은 순 (기본값)
- 금리 높은 순
- 한도 큰 순
- 한도 작은 순
- 기관명 가나다 순

**대상 그룹 필터 (Badge 버튼 토글)**:
- 전체 / 청년 / 신혼부부 / 다자녀 / 고령자

### 4-3. 상품 카드 개선

```
┌──────────────────────────────────────┐
│ ✓ 카카오뱅크 전세자금대출            │
│   카카오뱅크                         │
│                                      │
│   [변동]  [청년]                     │  ← 배지
│                                      │
│   3.45%~5.10%    최대 5억원          │  ← 범위 표시
│   (대표금리 3.45%)                   │
│                                      │
│   100% 비대면 신청, 간편 승인...      │
│                              [›]     │
└──────────────────────────────────────┘
```

**배지 색상 규칙**:
- `[변동]` → blue, `[고정]` → green, `[혼합]` → purple
- `[청년]` → orange, `[신혼부부]` → pink, `[다자녀]` → teal, `[고령자]` → gray

### 4-4. 추가 기능: 상품 비교 모드

**동작 방식**:
1. 상품 카드 우상단에 "비교추가" 체크박스
2. 최대 3개 선택 시 하단 플로팅 바 등장: "3개 상품 비교하기 →"
3. 비교 테이블 모달:

```
┌──────────────────────────────────────────────────────────────┐
│  상품 비교                                           [닫기]  │
├──────────────────┬───────────────┬──────────────────────────┤
│                  │ 카카오뱅크    │ 청년버팀목     │ NH 청년  │
├──────────────────┼───────────────┼───────────────┼──────────┤
│ 최저 금리        │ 3.20%         │ 1.80%         │ 3.30%    │
│ 최고 금리        │ 4.60%         │ 2.70%         │ 4.70%    │
│ 금리 유형        │ 변동          │ 고정          │ 변동     │
│ 최대 한도        │ 3억           │ 2억           │ 3억      │
│ 대상             │ 청년          │ 청년          │ 청년     │
│ 월 예상 납입     │ 89만원        │ 45만원        │ 90만원   │  ← 선택 매물 기준
│ 총 이자          │ 1,350만원     │ 540만원       │ 1,440만원│
└──────────────────┴───────────────┴───────────────┴──────────┘
│           [선택된 매물의 전세가 기준 자동 계산]              │
└──────────────────────────────────────────────────────────────┘
```

### 4-5. 추가 기능: 내 조건 맞춤 추천 배지

매물 선택 + 계산기 진입 시 상단에 배너:

```
┌────────────────────────────────────────────────────────┐
│  💡 아래 상품이 이 매물(3.2억 전세)에 적합합니다       │
│     [청년] 버팀목: 한도 2억, 금리 1.8% — 추천         │
└────────────────────────────────────────────────────────┘
```

조건: `selectedUnit.jeonsePrice >= maxAmount * 0.5` 이고 카테고리가 정부/공공이면 추천 표시.
